About the Seller

This portal is designed for managing product listings, tracking orders, and analyzing performance metrics for sellers on MauryaShop. It’s built with Django, Bootstrap, and a custom responsive design.

Seller Features

Product Management: Add, update, or remove product listings effortlessly.
Order Tracking: Stay updated with real-time order statuses.
Analytics: View reports and metrics for performance optimization.

Technologies Used

Frontend: HTML, CSS, Bootstrap
Backend: Python, Django
Database: SQLite (development)
Tools: Git, GitHub, Django Admin
